# Lab: HTML Form Elements

## Description
In this lab, you will learn how to create and use different types of form elements in HTML. Forms are used to collect user input and send data to a server.

## Steps

### Step 1: Create a New HTML File
1. Create a new file named `form.html` and add the basic HTML structure.

### Step 2: Add a Form Element
2. Create a `<form>` element to contain all input fields.

### Step 3: Add Input Fields
3. Add text fields for **Name** and **Email** with labels.

### Step 4: Add Radio Buttons
4. Add radio buttons for **Gender** selection.

### Step 5: Add Checkboxes
5. Add checkboxes for **Skills** selection.

### Step 6: Add a Dropdown List
6. Add a dropdown (`<select>`) for **Country** selection.

### Step 7: Add a Submit Button
7. Add a **Submit** button to send the form data.

## Summary
- Used `<input>` for text, email, radio, and checkboxes.
- Used `<select>` for dropdown.
- Used `<button>` to submit the form.
